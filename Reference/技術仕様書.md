# 技術仕様

- **Framework**: なし（純粋な HTML/JavaScript）
- **Styling**: Tailwind CSS (CDN版)
- **UI Components**: 
  - 進捗バー（DaisyUI等の流用または自作）
  - 合目に応じた富士山の状態変化（簡易的なCSSカラーチェンジ等）
- [cite_start]**Data Persistence**: LocalStorage API [cite: 15]
- **URL Handling**: URLSearchParams API
- **Backend/Realtime**: Supabase (Database & Auth) または Firebase
- **Animation**: 
  - `canvas-confetti` (お祝い演出用)
  - `Animate.css` (UI要素の動き)
- **Audio/Haptics**: 
  - Web Audio API (SE再生)
  - Navigator.vibrate() (バイブレーション)
